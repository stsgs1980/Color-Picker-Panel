# Варианты экспорта Color Picker Panel

## Обзор

Color Picker Panel можно экспортировать в различных форматах для использования в других проектах и средах.

---

## Сравнение вариантов

### 1. NPM пакет (React компонент)

**Что это:** Библиотека, которую устанавливают через `npm install` и импортируют в React проект.

```jsx
// Использование в чужом проекте
import { ColorPickerPanel } from '@my/color-picker';

function App() {
  const [color, setColor] = useState('#ff0000');
  return <ColorPickerPanel onChange={setColor} />;
}
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Простая установка | ❌ Только React проекты |
| ✅ Полный контроль | ❌ Нужно знать React |
| ✅ TypeScript поддержка | ❌ Зависимости (Tailwind, shadcn) |
| ✅ Tree-shaking | ❌ Вес бандла |

**Для кого:** React/Next.js разработчики

---

### 2. Web Component

**Что это:** Нативный HTML-элемент, работает в любом фреймворке.

```html
<!-- Использование где угодно -->
<color-picker-panel 
  primary="#ff0000" 
  on-change="handleChange">
</color-picker-panel>
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Работает везде (React, Vue, Angular, vanilla) | ❌ Сложнее разработка |
| ✅ Нативная поддержка браузеров | ❌ Ограниченный API |
| ✅ Инкапсуляция стилей (Shadow DOM) | ❌ Сложнее отладка |
| ✅ Независимость от фреймворков | ❌ Меньше инструментов |

**Для кого:** Команды с разными стеками, дизайн-системы

---

### 3. Figma Plugin

**Что это:** Плагин внутри Figma для дизайнеров.

```
Пользователь открывает Figma → Plugins → Color Picker Panel
                                    ↓
                          Выбирает цвет → Применяет к объекту
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Дизайнеры без кода | ❌ Только внутри Figma |
| ✅ Интеграция с объектами | ❌ Figma API ограничения |
| ✅ Сохранение в библиотеки | ❌ Другая экосистема |
| ✅ Бесплатное распространение | ❌ Сложнее обновления |

**Для кого:** UX/UI дизайнеры, дизайн-команды

---

### 4. Chrome Extension

**Что это:** Расширение браузера с popup панелью.

```
Клик на иконку → Color Picker Panel
                    ↓
              Выбор цвета → Копирование
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Всегда под рукой | ❌ Только Chrome/Edge/Firefox |
| ✅ Работает на любом сайте | ❌ Ограничения безопасности |
| ✅ Копирование в буфер | ❌ Нет интеграции с сайтом |
| ✅ Быстрый доступ | ❌ Размер ограничен |

**Для кого:** Фронтенд-разработчики, тестировщики

---

### 5. Standalone JS (Script tag)

**Что это:** Один JS файл, подключается через `<script>`.

```html
<!-- На любом сайте -->
<script src="color-picker-panel.min.js"></script>
<div id="color-picker"></div>

<script>
  ColorPickerPanel.init('#color-picker', {
    onChange: (color) => console.log(color)
  });
</script>
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Максимальная совместимость | ❌ Глобальные переменные |
| ✅ Нет сборки для пользователя | ❌ Нет tree-shaking |
| ✅ CDN подключение | ❌ Конфликты стилей |
| ✅ Простейшая интеграция | ❌ Большой файл |

**Для кого:** CMS (WordPress, Tilda), старые проекты

---

### 6. Electron App

**Что это:** Desktop приложение (Windows/Mac/Linux).

```
Установка .exe/.dmg → Приложение на рабочем столе
```

| Плюсы | Минусы |
|-------|--------|
| ✅ Офлайн работа | ❌ Большой размер (~150MB) |
| ✅ Доступ к системе | ❌ Нужно устанавливать |
| ✅ Автообновления | ❌ Сложная подпись/нотариализация |
| ✅ App Store / Microsoft Store | ❌ Разработка дороже |

**Для кого:** Дизайнеры, разработчики без браузера
